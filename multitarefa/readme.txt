Parâmetros dos jogos

Par ou Impar?

	timeout: tempo que expira a rodada (item)
	operators: operadores possíves ( +, -, /, * ) *melhor usar os 2 primeiros
	numOperators: numero de operadores que aparecem na expressão
	minValue: valor mínimo possível para o resultado
	maxValue: valor máximo possível para o resultado

Bolinhas caindo
	ballSpeed: velocidade das bolinhas
	ballDensity: quantidade de bolinhas

Triângulos
	bugSpeed: velodicade dos triângulos
	bugDensity: quantidade de triângulos simultâneos
	bugAngles: ângulos possíveis

Tabela
	tableSize: dimensões da tabela
	tableShowTime: tempo que a célula se mantem ativa
	tableTimeout: tempo que expira a rodada (item)
	
* o jogo está configurado para carregar a dificuldade do meio, o "level 2" (de 3). Isso corresponde à posição 1 do array (0,1,2) de levels.
